# THE TO-DO-APP

This is a simple to-do app built using JS. It allows users to add and delete tasks. The app also includes a feature to mark tasks as completed and store the inputs in local storage and get them back when you close the page.

## LANGUAGES

- JAVASCRIPT
- CSS
- HTML
